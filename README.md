# 🔐 PhishGuard – Phishing Detection System

PhishGuard is a cybersecurity web app that analyzes URLs and detects potential phishing threats using rule-based analysis.

## 🚀 Features
- URL scanning
- Risk scoring
- Explanation of threats
- Secure analysis

This project is for educational purposes only and is being improved with machine learning techniques.
