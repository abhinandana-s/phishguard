# 🔐 PhishGuard – Phishing Detection System
<img width="1897" height="912" alt="image" src="https://github.com/user-attachments/assets/f41d43a7-976c-451b-b00c-cd1cf3067257" />

PhishGuard is a cybersecurity web application that analyzes URLs and detects potential phishing threats using multi-factor rule-based analysis.

---

## 🚀 Features

- 🔍 URL risk analysis  
- 📊 Risk score generation (0–100)  
- 🧠 Threat breakdown:
  - URL Structure Risk  
  - Domain Risk  
  - Content Risk  
- 📈 Confidence level (Low / Medium / High)  
- ⚙️ Multi-engine scan simulation  
- 💡 Actionable recommendations  

---

## 🧠 Detection Techniques

- URL pattern analysis  
- HTTPS verification  
- Suspicious keyword detection  
- Domain structure analysis  

---

## 🖥️ Tech Stack

- Frontend: React + Tailwind CSS  
- Backend: FastAPI (Python)  

---

## 🔄 Future Improvements

- Machine learning-based phishing detection  
- Real dataset integration  
- Improved accuracy evaluation  

---

## ⚠️ Disclaimer

This project is built for educational and cybersecurity awareness purposes only.  
It does NOT perform any malicious activity.
